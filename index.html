  • darkonah

    A Heterogeneous Multi-Processing (HMP) kernel with default CPU frequency table, has great balance between battery usage and performance.

  • darkonahZ

    An Energy Aware Scheduling (EAS) kernel with default CPU frequency table, great battery usage without sacrificing performance.

  • darkonahX

    An Energy Aware Scheduling (EAS) kernel with Overclocked CPU & GPU frequency table, give more power and great performance.

  • Venus

    A pure CAF kernel based on Android 10 tag with many optimizations (Linux version 4.4.205). It's HMP kernel with Overclocked CPU & GPU frequency table. Update will be slow depends on update from CAF itself.
