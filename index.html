  • darkonah

    A Heterogeneous Multi-Processing (HMP) kernel with default CPU frequency table, has great balance between battery usage and performance.

  • darkonahZ

    An Energy Aware Scheduling (EAS) kernel with default CPU Frequency table, great battery usage without sacrificing performance.

  • darkonahX

    An Energy Aware Scheduling (EAS) kernel with overclock CPU Frequency, give more power and great performance.

  • Corvus

    A pure CAF kernel based on Android 10 tag with many optimizations. CPU frequency is overclock, but it's HMP kernel. Update will be slow depends on update from CAF itself.
